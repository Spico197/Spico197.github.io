How to Start Hexo

Posted on 2019-02-13 | Edited on 2019-07-28 | In note

Hexo is a static blog framework and we could use markdown syntax to render pages.

Installation

First, we have to get Node.js and Git ready.

Here are the two links:

  • Node.js
  • Git

Second, we should install hexo framework through npm commands.

Third, I also use a third-party theme, the NexT.

The Hexo version is 3.8.0 and the NexT version is 7.0.0 .
