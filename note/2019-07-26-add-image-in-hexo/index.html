Add Images into Hexo

Posted on 2019-07-26 | Edited on 2019-07-28 | In note

Hexo is not very image friendly. So we may not insert local images just like naive markdown.

  1. We should install a plugin

  1. Set _config.yml

  2. When we create a new post, hexo will generate a folder with the same name. We can put images or other things into this folder and add a tag like below into the post.

Or just insert images like ![alt](page1.png).

And finally, you could see these pictures in the local server.
